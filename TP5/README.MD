## Resumo

Aplicação web desenvolvida em Express.js com Pug e W3.CSS para consulta de uma base de dados de filmes. Permite listar e visualizar detalhes de filmes, atores e géneros, com navegação entre as três entidades. Os dados são servidos por um json-server a partir de um dataset de filmes.

## Lista de resultados

- Lista de filmes com título, ano e géneros
- Página de detalhe de cada filme com elenco e géneros clicáveis
- Lista de atores com número de filmes
- Página de detalhe de cada ator com os seus filmes
- Lista de géneros com número de filmes
- Página de detalhe de cada género com os seus filmes
