## Resumo

Este trabalho consiste no desenvolvimento de um pequeno servidor web em Node.js que permite consultar informação de uma escola de música.
O servidor recebe pedidos HTTP e, através da biblioteca Axios, obtém dados de uma API disponível localmente na porta 3000.
Com os dados recebidos, o servidor gera páginas HTML que apresentam listas de alunos, cursos e instrumentos.

## Lista de resultados

- Página inicial com links para as diferentes secções da aplicação.
- Lista de alunos com informação como nome, data de nascimento, curso, ano do curso e instrumento.
- Lista de cursos com designação, duração e instrumento associado.
- Lista de instrumentos existentes na escola de música.
- Servidor HTTP a escutar na porta 25000.
